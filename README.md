# Samavi ADM

Dashboard administrativo da **Samavi Tecnologia** — painel de controle do app PreçoJusto.

## O que é

Painel web com métricas de usuários, comerciantes B2B, financeiro e monitoramento de fraudes.

## Como rodar localmente

Abra o arquivo `index.html` diretamente no navegador — não precisa de instalação.

## Deploy

Hospedado no Vercel. Qualquer `git push` na branch `main` publica automaticamente.

## Estrutura

```
ADM/
└── index.html     # Dashboard completo
└── README.md
```

## Fluxo de atualização

```
Cursor (edita) → git push → Vercel (publica em ~1 min)
```
